Create a button for free downloadable WooCommerce content. You can use this shortcode to place this button in the product description or on the product page. Your visitors can access free content without missing a checkout.

== Description ==
After activating the plugin, the add to cart button will automatically change for download, where the product is free and can be downloaded. You can also place the button in the product description (or in the short description) so that the button is also available on the product page. To do this, you need to use the following shortcode: [woofreedownload]

How It Works?
The price of the product must be left blank. The price field cannot contain zero or any numeric value. If so, we will clear the price and update the product.

Shortcode: [woofreedownload]
It must be placed on the product side for proper operation. This can be in the short or detailed description of the product.

== Frequently Asked Questions ==
= How It Works? =
You will install, activate, and all free downloadable products will be available by skipping the checkout page.

= What shortcode to use? =
Shortcode: [woofreedownload]
It must be placed on the product side for proper operation. This can be in the short or detailed description of the product.

= Is the direct download button not displayed? =
Make sure you have not entered a value for the price of the product
